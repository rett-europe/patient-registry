European Rett Syndrome Patient Registry

The Rett Syndrome Patient Registry is a GDPR-compliant system designed to collect, store, and manage data about patients with Rett Syndrome. It enables legal guardians to onboard themselves and their patients via a public form without requiring authentication, with data initially stored in a staging database for validation.

Administrators can manage contacts, patients, and their relationships through a backoffice application with full CRUD capabilities. The system facilitates longitudinal data collection by sending secure, tokenized surveys to guardians, linking responses to specific patients over time.

It also provides a secure file upload feature for guardians to submit documents like genetic reports, ensuring files are encrypted and properly associated with patients. User management leverages the Microsoft 365 ecosystem for authentication and authorization, using Single Sign-On (SSO) and Role-Based Access Control (RBAC) to ensure secure access by authorized personnel.

Additionally, the system supports anonymized reporting and analytics through tools like PowerBI, maintaining high standards for data encryption and protection throughout.
